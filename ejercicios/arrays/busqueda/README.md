# Búsqueda

## B-01 - Sistema de productos con búsqueda y estadísticas

Una tienda tiene hasta 50 productos almacenados en un array.

Cada producto posee:

- nombre
- precio
- stock

Implementar:

1. Búsqueda secuencial por nombre.
2. Si se encuentra:
   - Mostrar los datos.
   - Informar si el stock es bajo (< 5).
3. Informar cuántos productos tienen precio mayor al encontrado.

## B-02 - Directorio de alumnos con búsqueda binaria

Un directorio almacena alumnos ordenados por legajo.

Cada alumno posee:

- legajo
- nombre
- edad
- promedio

Implementar:

1. Búsqueda binaria por legajo.
2. Si se encuentra:
   - Mostrar los datos.
   - Informar cuántos alumnos tienen promedio mayor.
   - Informar cuántos alumnos tienen promedio menor.

## B-03 - Agenda de facturas con búsqueda por rango

Una empresa posee facturas ordenadas por número.

Cada factura posee:

- número
- monto

Implementar:

1. Búsqueda binaria por número.
2. Búsqueda secuencial de todas las facturas cuyo monto esté dentro de un rango dado.
