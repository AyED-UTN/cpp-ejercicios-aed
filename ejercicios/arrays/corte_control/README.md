# Corte de Control

## C-01 - Resumen de ventas por vendedor

Una empresa registra las ventas del mes en un array ordenado por código de vendedor.

Cada venta posee:

- código de vendedor
- monto

Para cada vendedor mostrar:

- código
- cantidad de ventas realizadas
- monto total vendido
- monto promedio por venta

---

## C-02 - Informe de alumnos por curso

Se dispone de un array ordenado por curso.

Cada alumno posee:

- curso
- nombre
- promedio

Para cada curso mostrar:

- nombre del curso
- cantidad de alumnos
- promedio general
- cantidad de aprobados (promedio >= 6)
- cantidad de desaprobados

---

## C-03 - Facturación mensual por cliente

Se dispone de un array ordenado por código de cliente.

Cada factura posee:

- código de cliente
- número de factura
- monto

Para cada cliente mostrar:

- código de cliente
- cantidad de facturas
- monto total facturado

Al finalizar indicar qué cliente obtuvo el mayor monto total facturado.
