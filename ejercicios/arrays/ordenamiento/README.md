# Ordenamiento

## O-01 - Tabla de posiciones de fútbol

Una liga de fútbol almacena en un array los equipos participantes.

Cada equipo posee:

- nombre
- puntos acumulados

Se pide mostrar la tabla de posiciones ordenada de mayor a menor cantidad de puntos.

---

## O-02 - Ordenar personas por nombre y documento

Se dispone de un array de personas.

Cada persona posee:

- nombre
- documento

Implementar:

1. Ordenamiento por nombre en forma ascendente utilizando Selección.
2. Ordenamiento por documento en forma descendente utilizando Inserción.

---

## O-03 - Detección de vector ya ordenado

Implementar una función que determine si un array se encuentra ordenado ascendentemente sin modificarlo.

Utilizar dicha función para decidir si es necesario ordenar el array antes de realizar una búsqueda binaria.
