# Apareo

## A-01 - Catálogo unificado de productos

Dos sucursales poseen catálogos de productos ordenados por código.

Cada producto posee:

- código
- descripción

Generar un nuevo catálogo ordenado que contenga todos los productos de ambas sucursales.

---

## A-02 - Unión de conjuntos numéricos

Se dispone de dos arrays ordenados de números enteros.

Generar un tercer array ordenado que contenga todos los elementos de ambos arrays.

---

## A-03 - Alumnos de dos departamentos

Dos departamentos académicos poseen listados de alumnos ordenados por legajo.

Cada alumno posee:

- legajo
- nombre

Generar un único listado ordenado por legajo con los alumnos de ambos departamentos.
